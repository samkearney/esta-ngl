Property Definition¶

A property represents a single piece of data related to a device. A property definition provides information about how to read (and write, if supported) the data associated with a property. A property definition defines several attributes about a property.

Attributes¶

Definition attributes are part of the property definition; they apply to all instances of the property.

Class (Required)¶

A property’s class is a string identifying an individual property definition. This string must be unique within the Library or interface definition that contains the property definition.

Name (Required)¶

A string containing the human-readable name of the property. This attribute could be used as the translation key for properties of this class.

Description¶

A string that describes the purpose of a property in human-readable language.

Data Type (Required*)¶

A property’s data type specifies how to interpret the data stored with the property.

*Data Type is not required when the property definition includes the extend attribute. This attribute should be excluded, and will be ignored if extends is included.

The possible data types are:

number

A numeric value, either integral or floating point.

string

A UTF-8 string.

binary

Opaque binary data.

boolean

A boolean value (true or false).

enum

A set of meaningful names assigned to numeric values.

array

An array of values, all of which must be the same type.

Unit (Required*)¶

A property’s unit defines how the property’s value should be interpreted. Units are only well-defined for properties with data type number.

*Unit is only required when the Data Type attribute is included, and is defined as number. This attribute should be excluded, and will be ignored if extends is included.

Extends¶

Indicates the class of the property definition this definition extends.

Markup¶

  • Tag name: propertydef

  • Attributes:

    • class: Class (Required)

    • name: Name (Required)

    • description: Description

    • datatype: Data Type (Required*)

    • unit: Unit (Required*)

    • extends: Extends

Example:

<propertydef
  class="strobe-frequency"
  name="Strobe Frequency"
  description="The frequency of a strobe shutter in Hertz"
  datatype="number"
  unit="hertz" />
