Device Definition¶

A device definition defines the behavior of a device, which is some type of equipment that is configurable. The term “device” in this standard is typically used to refer to a single model of equipment, and “device instance” refers to a single unit of that model.

Attributes¶

Class (Required)¶

A device definition’s class is a string identifying an individual interface definition. This string must be unique within the Library.

Markup¶

XML

JSON

  • Tag name: devicedef

  • Attributes:

    • class: Class (Required)

Example:

<devicedef class="es1">
  <interface class="org.esta.device.1/device" alias="device" friendlyname="Device">
    <override refalias="device-id" attribute="value">com.etcconnect.es1.1</override>
    <override refalias="device-manufacturer" attribute="value">ETC Inc.</override>
    <override refalias="device-model" attribute="value">ES1</override>
  </interface>
</devicedef>
