Intensity Properties¶

Intensity¶

A continuous intensity value represented in percent of maximum brightness.

Definition¶

Class

intensity

Qualified Class

org.esta.intensity.1/intensity

Name

Intensity

Description

Represents an emitter intensity as a percentage of its maximum brightness.

Data Type

number

Unit

percent

Binary Intensity¶

A binary intensity value used for non-dimming light sources.

Definition¶

Intensity properties control the light intensity of a light-emitting device.

Class

binary-intensity

Qualified Class

org.esta.intensity.1/binary-intensity

Name

Binary Intensity

Description

Represents an emitter intensity as an on/off state where “true” indicates “on”.

Data Type

boolean
