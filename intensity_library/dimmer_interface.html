Dimmer Interface¶

The dimmer interface represents a simple dimmer controlling an intensity.

Definition¶

Class

dimmer

Qualified Class

org.esta.intensity.1/dimmer

Name

Dimmer

Description

A simple dimmer controlling an intensity.
