Interface Instance¶

An interface instance is a single addressable instance of a Interface Definition.

Attributes¶

Instance attributes are per-interface-instance; they can vary between instances of the interface.

Id¶

The id is a unique string identifying this interface instance. The string must be unique within a device definition.

Markup¶

XML

  • Tag name: interface

  • Attributes:

    • type: Type

    • id: Id

Example:

<interface type="esta.interface.1" id="my-interface-instance" />
