Property Value¶

A property value represents the global value of a static property. It is a way to assign values to static properties defined in an interface definition when instantiating that interface.

Attributes¶

Id¶

The id of the property instance which is being assigned a value.

Markup¶

XML

  • Tag name: propertyvalue

  • Attributes:

    • id: Id
