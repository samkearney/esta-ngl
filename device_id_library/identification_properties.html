Identification Properties¶

Device ID¶

A globally unique identifier for a model of device. This identifier should follow a single device model for its lifetime and should not change with new firmware or hardware revisions of the device.

Definition¶

Class

device-id

Qualified Class

org.esta.identification.1/device-id

Name

Device ID

Description

A globally unique identifier for a device model.

Data Type

string

String Constraints

Follows the convention of a Qualified Device ID.

Definition Markup¶

<propertydef
  class="device-id"
  name="Device ID"
  description="A globally unique identifier for a device model."
  datatype="string" />

{
  "udrtype": "propertydef"
  "class": "device-id",
  "name": "Device ID"
  "description": "A globally unique identifier for a device model.",
  "datatype": "string"
}
